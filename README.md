# 🚀 GitHub Explorer

A responsive GitHub profile search application built with **React**, **Vite**, and the **GitHub REST API**. Search any GitHub username to view profile details and public repositories instantly.

## ✨ Features

- 🔍 Search GitHub users
- 👤 View profile information
- 📂 Display public repositories
- ⚡ Fast and responsive UI
- ❌ Error handling for invalid usernames

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS
- GitHub REST API
- Framer Motion
